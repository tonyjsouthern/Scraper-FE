F.A.Q.

What does this application do?

This application takes a given website and checks to see if the following companies tracking scripts are present. If a tracking script is found a "true" value will be returned, otherwise the value will default to false.

  1. Salesfusion
  2. Marketo
  3. ActOn
  4. Google
  5. Click Dimensions
  6. Pardot
  7. Hubspot

How does this application check for tracking scripts?

This application uses http and https methods to load a websites HTML. It then scans the entire websites HTMl for common keywords found in tracking scripts.

How can I suggest a tracking script check be added?

You can contact tonyjsouthern@gmail.com with the tracking script you would like to be added. This will be considered and possible added. If it is added I will reach out and let you know.

How throrough is this application?

This application only checks the websites static HTML. It is possible through some vendors to inject the scripts, if this is the case, this application will read false.
